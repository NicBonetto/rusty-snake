# rusty-snake
An implementation of the original Snake game in Rust.


### How To Play
*Requirements:* 
  &#x2714; Git
  &#x2714; Rust
  
Run the following commands to install locally
```
$ cd /path/to/installation
$ git clone 
```

Run the following commands to run the game
```
$ cargo build
$ cargo run
```

Enjoy &#128013;
